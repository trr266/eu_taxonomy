# Data repository EU-Taxonomy Regulation

*as of: 09 November 2022*


This repository contains the data for [Gro√ükopf, Sellhorn & Wagner (2022)](https://research.owlit.de/document/bf0947b5-36c1-3f5d-a8a9-52b02a347050). 
On a continuous basis, we collect the KPI mandated by the EU-Taxonomy as well as additional information.

Explanation of columns:

- `excl`: if the company is excluded and if so, why:
    - `SR_miss`: missing sustainability report
    - `FIRE`: the company is either a financial, insurance or real estate company
    - `FYE`: due to a different fiscal year end, data is not available yet
    - `data_missing`: data is missing
- `loc`: where the Taxonomy-related information is published:
    - `AR`: annual report
    - `SR`: sustainability report


The data can be downloaded in a separate csv file.
